# TP_IsaacPBouchard

J'ai trouvé mes données sur le site de la ville de Montréal dans les données ouvertes du secteur de la santé. Je les ai ensuite téléchargées dans un dossier de mon ordi pour les read.csv dans R, j'ai appelé ces données "data". J'ai choisi de prendre des données d'inspections préventives en salubrité du Service de l'habitation puisque l'analyse de celles-ci pourraient notamment nous indiquer s'il y a possibilité de sinistres pour un tel bâtiment. Cela peut donc être utile pour modéliser le risque en assurance habitation.
